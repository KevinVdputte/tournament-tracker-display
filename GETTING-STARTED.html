Tournament Tracker Display - Getting Started

This is a simple guide to help you run the Tournament Tracker application on your computer.

Prerequisites

Before you can run this application, you need to install Node.js on your computer.

Download Node.js

Download the "LTS" version (recommended for most users) and follow the installation instructions.

Windows Users

Once Node.js is installed, you can start the application by:

  1. Double-click the start-app.bat file in the application folder
  2. A command window will open and the application will start automatically
  3. When the application is running, open your web browser and go to: http://localhost:5173
If you see a Windows security warning, click "More info" and then "Run anyway". This is just Windows being cautious about running scripts.

Mac Users

Once Node.js is installed, you can start the application by:

  1. Open Terminal (press Cmd+Space, type "Terminal", press Enter)
  2. Navigate to the application folder using the cd command
  3. Run the start script by typing: ./start-app.sh
  4. When the application is running, open your web browser and go to: http://localhost:5173

Troubleshooting

If you have any issues running the application:

  • Make sure Node.js is properly installed
  • Try restarting your computer after installing Node.js
  • Check the INSTALL.md file for more detailed instructions

For more detailed instructions, please refer to the INSTALL.md file included with this application.
